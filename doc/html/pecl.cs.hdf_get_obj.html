Name

hdf_get_obj — Return the HDF data set node at a named location.

Description

resource hdf_get_obj(hdf,  
 name); 
resource hdf;
string name;
 

Returns an HDF resource representing node name in the parent HDF resource hdf, or null if name does not exist.

Note

This is similar to hdf_get_node except that if the node name does not exist it is not automatically created.

Example 13. hdf_get_obj example

			
<?php
$hdf = hdf_init();
print_r(hdf_get_obj($hdf, 'data.node'));
?>
