Name

hdf_get_value — Return the value of a node in the data set.

Description

string hdf_get_value(hdf,  
 name,  
 defval); 
resource hdf;
string name;
string defval;
 

Returns the value of node name in HDF resource hdf, or defval if name does not exist.

Example 9. hdf_get_value example

			
<?php
$hdf = hdf_init();
hdf_set_value($hdf, 'data.node', 'my value');
print hdf_get_value($hdf, 'data.node', 'default value');
?>
