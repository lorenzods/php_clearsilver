Name

hdf_obj_name — Return the name of a node.

Description

string hdf_obj_name(hdf); 
resource hdf;
 

Returns the node name of an HDF resource hdf.

Example 14. hdf_obj_name example

			
<?php
$hdf = hdf_init();
$node = hdf_get_node($hdf, 'data.node');
print hdf_obj_name($node); // outputs 'node'
?>
