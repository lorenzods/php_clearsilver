Name

hdf_remove_tree — Delete a subtree of an HDF dataset.

Description

bool hdf_remove_tree(hdf,  
 name); 
resource hdf;
string name;
 

Removes the node name in HDF resource hdf, returning true on success or false if an error is raised.

Example 19. hdf_remove_tree example

			
<?php
$hdf = hdf_init();
hdf_set_value($hdf, 'data.node', 'my value');
print hdf_write_string($hdf);

print "\nREMOVING NODE...\n";
hdf_remove_tree($hdf, 'data.node');
print hdf_write_string($hdf);
?>
